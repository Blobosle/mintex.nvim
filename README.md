
# mintex.nvim
Minimal LaTeX compilation for Neovim.

<img width="450" height="400" alt="mintex" src="https://github.com/user-attachments/assets/5121fdb8-f99b-4d1b-97ae-958ae1ad0ff7" />


## Usage
To compile the current .tex file and open it as a LaTeX pdf.
```lua
:Latex
```

## Setup
```lua
use 'Blobosle/mintex.nvim'
```
